
[[История команд]]

- #### ls (list)

Команда `ls` используется для просмотра содержимого в папке.
Существуют различные атрибуты к данной команде, например, -l для более детальной информации о папке

- #### cat (concatenate)

Команда `cat` позволяет вывести на экран текст созданный командой nano 

- #### cd (come directory)

Команда `cd` используется для работы с выбранной папкой, переданной в качестве аргумента

- #### pwd (print working directory)

Команда `pwd` печатает текущую рабочую среду (папку), в который мы находимся

- #### mkdir (make directory)

Команда `mkdir` создает папку с названием, переданной в качестве аргумента 

- #### file 

Команда `file + название файла` показывает тип файла. Например, для текстового документа nano с названием kali.txt, после применения программы file, вывод будет:

> **kali.txt: ASCII text**
